# NLP Text Classification Project

## 📌 Overview

This project focuses on building an efficient **Natural Language Processing (NLP)** based text classification system. The system analyzes input text and classifies it into predefined categories using machine learning models.

The project also includes a **GUI-based application** built using Streamlit for real-time predictions.

---

## 🎯 Objective

* To preprocess and analyze textual data using NLP techniques
* To train and compare multiple classification models
* To evaluate models using accuracy, precision, recall, and F1-score
* To select the best-performing model (accuracy ≥ 95%)
* To deploy the model using an interactive GUI

---

## 🛠️ Technologies Used

* Python 
* Scikit-learn
* Pandas & NumPy
* NLP (TF-IDF / CountVectorizer)
* Streamlit (GUI)

---

## 💻 GUI Application

The Streamlit-based GUI allows users to:

* Enter text input
* Click on the **Predict** button
* View classification results instantly

---

## 📊 Model Evaluation

Multiple models were trained and evaluated using:

* Accuracy
* Precision
* Recall
* F1-score

The best-performing model with accuracy ≥ 95% was selected for deployment.
